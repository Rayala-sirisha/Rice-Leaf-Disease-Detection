**Rice Leaf Disease Detection**
**Overview**

This project uses Deep Learning and Computer Vision techniques to detect and classify rice leaf diseases from image data. The model helps identify diseases at an early stage, supporting farmers in improving crop health and productivity.

**Objectives**
Detect rice leaf diseases automatically.
Classify leaf images into different disease categories.
Support precision agriculture through AI-based disease detection.

**Dataset**
The dataset contains rice leaf images belonging to:
Bacterial Leaf Blight
Brown Spot
Leaf Smut

**Technologies Used**
Python
TensorFlow / Keras
NumPy
Pandas
Matplotlib
OpenCV
Deep Learning

**Methodology**
Data Collection
Data Preprocessing
Data Augmentation
Model Training
Model Evaluation
Disease Prediction

**Results**
Training Accuracy: 97.83%
Validation Accuracy: 84.06%

**Applications**
Smart Farming
Precision Agriculture
Crop Health Monitoring
Early Disease Detection
